Breve descrição do jogo/fase;
O player controla o personagem com o nome de guy, um homem imortal que terá que correr da única coisa que supostamente poderia acabar com sua vida, um caracol imortal. Você terá que juntar peças de um foguete para fugir o mais rápido o possível.

Instruções básicas para executar o projeto.
Clone esse repositório do Github em seu PC
Utilize a Unity 2022.3 LTS para execução 
Aperte play na cena única da Unity e teste o jogo
Para fazer uma Build, basta clicar em File > Build Settings > Build > Clean Build
